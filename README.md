# Sales Forecasting Dashboard – Power BI

## Overview
Sales forecasting dashboard analyzing 2014–2017 retail data with 6-month ETS forecast.

## Key Features
- KPI cards: Sales, Profit, Margin, Quantity
- Category & regional analysis
- Monthly trend with forecast (ETS model)
- Seasonal decomposition
- MAPE & MAE accuracy benchmarks

## Tools Used
Power BI Desktop
DAX
ETS Exponential Smoothing

## Project Outcome
Developed an interactive Power BI dashboard to analyze historical sales data, identify trends, and generate future sales forecasts for business decision-maling.
